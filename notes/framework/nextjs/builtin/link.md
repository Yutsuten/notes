## Overview

Enable client-side transitions between routes.

## Usage

```js
import Link from 'next/link'
```

| Prop | Description |
| --- | --- |
| `ref` | **[Required]** The path or URL to navigate to. |

## Examples

```html
<Link href="/">Home</Link>
```
