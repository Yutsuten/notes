## Navigation

Methods available in an instance of `webdriver`.

| Method | Description |
| --- | --- |
| `get(url)` | Navigate to a URL. |
| `refresh()` | Refresh the current page. |
| `back()` | Press the browser’s back button. |
| `forward()` | Press the browser’s forward button. |
