## Usage

See a full list of keyboard keys available [here](https://developer.mozilla.org/en-US/docs/Web/API/UI_Events/Keyboard_event_key_values).

## Examples

Moving the viewport:

```javascript
await page.keyboard.press('PageDown');
await page.keyboard.press('Home');
```
