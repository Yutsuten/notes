## Create the PHP file

```php
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width">
        <title>Hello World page</title>
    </head>
    <body>
        <?php echo 'Hello World!' ?>
    </body>
</html>
```

## Move to apache folder

```bash
mv hello.py /var/www/html/
firefox localhost/hello.php
```
