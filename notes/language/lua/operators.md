## Arithmetic

| Operator | Description    |
| -------- | -------------- |
| `+`      | Addition       |
| `-`      | Subtraction    |
| `*`      | Multiplication |
| `/`      | Float division |
| `//`     | Floor division |
| `%`      | Modulo         |
| `^`      | Exponentiation |
| `-`      | Unary minus    |

## Relational

| Operator | Meaning                  |
| -------- | ------------------------ |
| `==`     | Equals                   |
| `~=`     | Not equals               |
| `>`      | Greater than             |
| `<`      | Less than                |
| `>=`     | Greater than or equal to |
| `<=`     | Less than or equal to    |

## Logical

| Operator | Description |
| -------- | ----------- |
| `and`    | AND         |
| `or`     | OR          |
| `not`    | NOT         |

## Bitwise

| Operator | Description          |
| -------- | -------------------- |
| `&`      | Bitwise AND          |
| `\|`     | Bitwise OR           |
| `~`      | Bitwise exclusive OR |
| `>>`     | Right shift          |
| `<<`     | Left shift           |
| `~`      | Unary bitwise NOT    |
