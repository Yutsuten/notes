## Common types

```c
char         // %c    8 bits
short        // %hi   16 bits
int          // %i %d 16 bits
long         // %li   32 bits
long long    // %lli  64 bits
float        // %f    32 bits
double       // %lf   64 bits
long double  // %Lf   128 bits

signed <type>   // Force type to be signed
unsigned <type> // Force type to be unsigned
```
