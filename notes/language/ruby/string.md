## Basic

```ruby
my_string = "Taro"

my_string.length
my_string.reverse
my_string.upcase
my_string.downcase
my_string.capitalize
my_string.gsub(/s/, "th")  # Global substitution
my_string.include? "substring"

my_string << "concatenate me"

words_array = my_string.split(" ")
```
