## Usage

| Text style                   | Syntax                         |
| ---------------------------- | ------------------------------ |
| **bold**                     | `**bold**` `__bold__`          |
| *italic*                     | `*italic*` `_italic_`          |
| ***bold and italic***        | `***text***` `___text___`      |
| ~~strikethrough~~            | `~~strikethrough~~`            |
| [Link](https://example.com/) | `[Link](https://example.com/)` |
| Image                        | `![title](IMAGE_URL)`          |
