## Basic

### Built-in functions

```js
typeof myVar;  // 'string', 'number', 'boolean', 'object', 'undefined'
isNaN(myVar);
```

#### Type conversion

```js
const num = parseInt('10');
const dec = parseFloat('3.1416');
const bool = Boolean(0);  // !!0 - true for empty lists and objects!
```
