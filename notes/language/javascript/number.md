## Basic usage

```js
mynum = 1000;        // Decimal notation
mynum = 1e3;         // Exponential notation (same as 1 * 10^3 = 1000)
mynum = 0xff;        // Hexadecimal notation
mynum = 0b11111111;  // Binary notation
```
