## Open files as arguments

```vim
:args notes/**/*.md
```

### Argument navigation

| Command | Description |
| --- | --- |
| `:ar` | List arguments or define new argument list. |
| `:n` `:N` | Next or previous argument. |
| `:fir` `:la` | Jump to first or last argument. |
| `:argu` | Jump to argument NUM, defaults to current. |
