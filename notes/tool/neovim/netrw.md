## Open in a new split

```shell
o   # Open file in horizontal split
v   # Open file in vertical split
```

## Change current directory

```shell
cd    # May be needed to mark target (see bellow)
```

## File manipulation

```shell
:Ex
mf    # Mark file
mF    # Unmark file
mu    # Unmark all
mt    # Mark target

mc    # Copy marked files to target
mm    # Move marked files to target
```
