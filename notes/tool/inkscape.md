## Crop image

1. `R`: Create a rectangle on the object
2. `S`: Move and resize the rectangle, select both objects
3. `Object > Clip > Set`: Apply the crop
4. `Ctrl+Shift+R`: Fit canvas to image
5. `Ctrl+Shift+E`: Export image
