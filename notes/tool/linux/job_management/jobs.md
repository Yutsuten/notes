## Usage

### Bash

```shell
jobs OPTIONS
```

| Option | Description |
| --- | --- |
| `-l` | Include job ID. |
| `-r` | Running jobs. |
| `-s` | Suspended jobs. |
| `-p` | Process Group ID (PGID) only. |

### Fish

```shell
jobs OPTIONS
```

| Option | Description |
| --- | --- |
| `-c` | Prints the command name for each process in jobs. |
| `-p` | Prints the process ID for each process in all jobs. |
