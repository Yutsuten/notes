## Shortcuts

### Basic actions

| Shortcut | Description |
| --- | --- |
| `Enter` `→` `L` | Open file |
| `←` `H` | Back to parent directory |
| `Space` | Select |
| `dd` | Cut |
| `dD` | Delete |
| `pp` | Paste |
| `F7` | Create new folder |

### Tabs

| Shortcut | Description |
| --- | --- |
| `Ctrl+n` | New tab |
| `Tab` | Switch to next tab |
| `Shift+Tab` | Switch to previous tab |
