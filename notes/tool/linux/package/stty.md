## Usage

Print or change terminal characteristics.

```shell
stty SETTING
```

| Setting | Description |
| --- | --- |
| `cols N` | Tell the kernel that the terminal has N columns. |
| `rows N` | Tell the kernel that the terminal has N rows. |
