## Usage

List, test and extract compressed files in a ZIP archive.

```shell
unzip OPTIONS file.zip -d DEST
```

| Option | Description |
| --- | --- |
| `-l` | List filenames. |

If the encoding of file names are not correcty detected,
use `unar` instead.
