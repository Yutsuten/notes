---
title: Unar
---

Objective-C tools for uncompressing archive files.

## Usage

```shell
unar OPTIONS ARCHIVE
```

| Option | Description |
| --- | --- |
| `-e` | The encoding to use for filenames. Use `help` to get a list. |

Check if the filenames are properly encoded with the `lsar` command.

```shell
lsar ARCHIVE
```
