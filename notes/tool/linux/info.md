## Hardware / system information

Some information are in files and can be seen using tools like `cat`.

| Information | File | Command |
| --- | --- | --- |
| OS | `/etc/os-release` | `hostnamectl` |
| Network | | `hostname -i` `ip -o address` `curl ifconfig.me` |
| CPU | `/proc/cpuinfo` | |
